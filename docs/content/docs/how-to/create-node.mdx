title: Create Node
description: Learn how to create container, text, and image nodes using @takumi-rs/helpers
---

Nodes are the foundation of building layouts with Takumi. The `@takumi-rs/helpers` package provides convenient functions for creating different types of nodes: containers, text, and images.

## Prerequisites

Make sure you have installed the helpers package:

```bash
npm install @takumi-rs/helpers
```

## Container Nodes

A container node is used to group other nodes and arrange them in a layout. Use the `container` helper:

```typescript
import { container } from "@takumi-rs/helpers";

const root = container({
  // extra styling goes here
  children: [
    // child nodes go here
  ],
});
```

## Text Nodes

A text node displays text. Use the `text` helper:

```typescript
import { text } from "@takumi-rs/helpers";

const helloText = text("Hello, Takumi!", {
  // extra styling goes here
});
```

You can add this text node as a child of a container:

```typescript
const root = container({
  // extra styling goes here
  children: [helloText],
});
```

## Image Nodes

An image node displays an image from a URL or local path. Use the `image` helper:

```typescript
import { image } from "@takumi-rs/helpers";

const logo = image("/logo.png", {
  // extra styling goes here
});
```

You can combine all node types to build layouts:

```typescript
import { container, text, image } from "@takumi-rs/helpers";

const layout = container({
  // extra styling goes here
  children: [
    image("/logo.png"),
    text("Welcome to Takumi!"),
  ],
});
```

For details on customizing appearance and layout, see [How to Style a Node](/docs/how-to/styling-node).
