Nodes are the foundation of building layouts with Takumi. The `@takumi-rs/helpers` package provides convenient functions for constructing different types of nodes: containers, text, and images.

## Prerequisites

Make sure you have installed the `@takumi-rs/helpers` package:

```bash
npm install @takumi-rs/helpers
```

## Container Nodes

A container node is used to group other nodes and arrange them in a layout. Use the `container` function:

```typescript
import { container } from "@takumi-rs/helpers";

const root = container({
  // extra styling goes here
  children: [
    // child nodes go here
  ],
});
```

## Text Nodes

A text node displays text. Use the `text` function:

```typescript
import { text } from '@takumi-rs/helpers';

const helloText = text('Hello, Takumi!', {
  // extra styling goes here
});
```

You can add this text node as a child of a container:

```typescript
const root = container({
  // extra styling goes here
  children: [helloText],
});
```

For details on loading fonts and rendering styled text, see [how to render styled text](/docs/how-to/rendering-text).

## Image Nodes

An image node displays an image from a URL or local path. Use the `image` function:

```typescript
import { image } from '@takumi-rs/helpers';

const logo = image('/logo.png', {
  // extra styling goes here
});
```

You can combine all node types to build layouts:

```typescript
import { container, text, image } from '@takumi-rs/helpers';

const layout = container({
  // extra styling goes here
  children: [
    image('/logo.png'),
    text('Welcome to Takumi!'),
  ],
});
```

For details on loading and rendering images, see [how to load and render images](/docs/how-to/load-image).

For details on customizing appearance and layout, see [how to customize appearance and layout](/docs/how-to/styling-node).
